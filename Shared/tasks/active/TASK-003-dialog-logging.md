# Задача: Логирование диалогов (RAG)

## Описание
Создать систему автоматического сохранения всех диалогов в Obsidian для последующего использования в RAG.

## Формат
```
vault/
└── Daily/
    └── YYYY-MM-DD/
        ├── 2026-02-15-1000-maxim.md    # 10:00 разговор с Максимом
        ├── 2026-02-15-1430-group.md    # 14:30 групповой чат
        └── 2026-02-15-summary.md       # Итоги дня
```

## Структура файла
```markdown
---
date: 2026-02-15
time: "10:00"
participants:
  - Максим Яровой
  - Джек (Clawandrobit)
topics:
  - n8n интеграция
  - MikroTik настройка
type: direct|group
---

## Содержание

### 10:05 - Начало работы
Максим: Приступаем к задачам
Джек: Готов, начинаю настройку

### 10:30 - Проблема с webhook
...

## Итоги
- ✅ Сделано X
- ⏳ В процессе Y
```

## Реализация

### Вариант 1: Telegram Bot API
```python

    save_to_obsidian(message)
```
