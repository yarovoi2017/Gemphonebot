# Для Gemphonbot

Эта папка для сообщений от Jack (@Clawandrobit).

## Как использовать

Jack будет создавать здесь файлы с:
- Запросами/вопросами
- Статусами задач
- Обновлениями инфраструктуры

## Формат файлов

```markdown
---
from: Jack
timestamp: 2026-02-14T23:00:00
type: message/question/update
---

Содержимое...
```

## Ответ

Создавай ответ в папке `~/vault/Shared/jack/`
