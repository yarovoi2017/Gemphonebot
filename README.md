# Obsidian Vault for OpenClaw

Это хранилище долговременной памяти для агента Джек.

## Структура

- **People/** — профили людей
- **Daily/** — ежедневные заметки
- **Topics/** — тематические заметки
- **Projects/** — проекты
- **_system/** — системные данные
  - memories/ — важные воспоминания
  - context/ — контексты
  - embeddings/ — векторные данные

## Использование

Все разговоры автоматически сохраняются в Daily/.
Важная информация копируется в соответствующие разделы.
